# LEGATO-Theory-Platform
LEGATO Theory Platform es una plataforma interactiva para aprender teoría musical de forma práctica. Incluye piano interactivo, acordes, escalas, círculo de quintas, metrónomo y herramientas de estudio. Ideal para estudiantes, músicos y profesores que desean comprender y practicar los fundamentos de la música. Diseñada por el profesor Freddy Lara.
